Playground in Scala
===================

Prerequisites
=============

  The following packages have to be installed correctly.

    - Java 1.6.x
    - Scala 2.10.2
    - scalatest_2.10-1.9.1
    - sbt 0.12.2

How to run
==========

   $ cd scala && ls
   build.sbt src

   $ sbt
   > run 5
   > exit

   Run the tests.
   $ sbt test
