Playground in Scala
===================

Prerequisites
=============

  The following packages have to be installed correctly.

    - Java 1.6.x
    - Scala 2.11.1
    - scalatest_2.11-2.2.0
    - sbt 0.13.5

How to run
==========

   $ cd scala && ls
   build.sbt src

   $ sbt
   > run 5
   > exit

   Run the tests.
   $ sbt test
